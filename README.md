# 🕋 Hajj Crowd Safety Prediction System 

## 📌 Overview

A machine learning system designed to predict high-risk zones during Hajj using factors like age, health condition, crowd density, and environmental data.
An interactive dashboard visualizes risks in real time to support better decision-making.

---
## 📊 Data
The dataset used in this project is synthetic and simulates real Hajj pilgrim data, similar to what might be collected from a Nusk card.
including personal information, health measurements, and other relevant details for risk assessment.
* Pilgrim demographics (age, health)
* Crowd density and movement
* Environmental conditions (temperature, humidity)

---
## 🤖 Models

* Random Forest (best performance)
* Neural Network (for comparison)

---
## 📈 Results

* Random Forest achieved high accuracy in classifying risk levels
* Pilgrims segmented into 3 groups:

  * High-risk (elderly, crowded routes)
  * Medium-risk
  * Low-risk (young, less crowded)

---

## 🗺️ Dashboard
* Interactive map showing high-risk areas
* Real-time visualization of crowd congestion
